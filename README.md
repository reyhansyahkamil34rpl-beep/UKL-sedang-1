 Deskripsi Program

Program ini dibuat untuk menghitung faktorial dari sebuah bilangan bulat yang dimasukkan oleh pengguna.
Faktorial merupakan operasi matematika yang menghitung hasil perkalian berurutan dari 1 hingga angka tersebut:
Contoh:
5! = 5 √ó 4 √ó 3 √ó 2 √ó 1 = 120

Program ini juga menampilkan proses perkalian secara lengkap pada layar.
 Fitur Program
 Input bilangan bulat dari pengguna
 Validasi bilangan (tidak boleh negatif)
 Menghitung faktorial menggunakan perulangan for
 Menampilkan proses perkalian faktorial
 Menampilkan hasil akhir perhitungan

Cara Menjalankan Progra
Jalankan compiler Java (misal melalui terminal/IDE)
Compile:
javac UKLsedang1.java
Jalankan:
java UKLsedang1
Masukkan bilangan bulat ketika diminta
Contoh Output
Masukkan sebuah bilangan bulat: 
5
1 x 2 x 3 x 4 x 5 = Faktorial dari 5 adalah: 120
Jika bilangan negatif:
Masukkan sebuah bilangan bulat: 
-3
Tidak dapat menghitung faktorial
